Backend memakai dua pola akses Prisma: prisma.<model>. untuk query standar, dan tx.<model>. di dalam prisma.$transaction(async (tx) => {...}). Kedua pola sudah dihitung pada angka 198 model "used".

4. Kesimpulan & Rekomendasi

✓ Yang Sudah Baik

  • Schema Prisma tersinkron 99.7% dengan DB
  • 198 model dipakai aktif di backend
  • Struktur ESG generasi baru terpakai komprehensif di esg.main.ts
  • Implicit M2M (_group_users_permissionTojabatan) ter-handle dengan benar

⚠ Yang Perlu Tindakan

  • Tabel error_logs di DB tidak punya model — investigasi atau drop
  • 1 model bermasalah karena nama tabel terpotong PostgreSQL (≤63 char limit)
  • 92 model Prisma jadi technical debt — sisa Laravel + ESG generasi-1
  • Ada beberapa pasang model duplikat dengan nama mirip (gen-1 vs gen-2)
